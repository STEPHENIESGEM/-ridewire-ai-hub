# RideWire AI Hub

Short description:
RideWire AI Hub is a web app that integrates AI-driven features (RAG, model-driven responses) with a game-like UI.

Quickstart (local)
1. Install Node.js LTS: https://nodejs.org
2. Clone the repo:
   git clone https://github.com/STEPHENIESGEM/-ridewire-ai-hub.git
   cd -ridewire-ai-hub
3. Install dependencies:
   npm install
4. Run locally:
   npm run dev
5. Open http://localhost:3000 (or the address printed in the terminal).

Testing
- E2E tests:
  npm run test:e2e
- Unit tests:
  npm test
