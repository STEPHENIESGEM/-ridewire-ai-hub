## MULTI-AI BRAIN ARCHITECTURE (What You're Building)

### The Three-Agent System

**Specialist Agents:**
- Engine Diagnostics Agent
- Electrical Systems Agent
- Transmission Agent
- Emissions Agent
- Suspension & Safety Agent

**Coordinator Agent:**
- Receives inputs from all specialists
- Merges independent reasoning into one ranked diagnosis
- Visible as the "pulsating brain" in the HUD

**Supervisor Agent:**
- Checks all recommendations for dangerous procedures
- Flags risky repairs before they happen
- Safety-first guardrail

### Input Data (For Simulation First)

- OBD-II / CAN Bus Data (simulated)
- Diagnostic Trouble Codes (DTCs) - examples like P0300
- Live Sensor Values - RPM, temperature, voltage
- Vehicle Info - make, model, year, trim, mileage
- Symptom Description - "rattling noise," "hard shift," etc.
